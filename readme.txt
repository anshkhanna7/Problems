RectangleOverlap: A function to check whether two rectangles overlap or not. Uses range overlap as a condition

SpellCheck: Code to generate possible matches of a misspelt word from an input dictionary. Handles cases of edit distance 1 and 2 individually.

TwitterAnalysis: Uses twitter api to get following of a popular account and clusters the accounts on the basis of length of their tweets using 2 parameters, mean and variance.
